### 개발 환경
---
**디바이스**
- MacBook Air M1(2020)
- Mac mini M2(2023)

**IDE**
- Xcode : Version 15.2 / 시뮬레이터 iOS (15.0)
- Android Studio : Giraffe (2022.3.1)

**리액트 네이티브**
- react-native : 0.73.3
- JDK : Java 17

### 프로젝트 설정
---
1. 소스코드 클론
2. 모듈 설치 : yarn install
3. 실행
   - 안드로이드 : npm run android
   - iOS
      - cd ios && pod install
      - cd .. && npm run ios
